# Meetup Frontend v2

Modern React frontend built with Vite, Tailwind CSS, and shadcn/ui.

## Features

- **Public Events Page**: Browse all meetups without authentication
- **Authentication**: Login and Register with form validation
- **Meetup Management**: Create, edit, delete meetups
- **RSVP System**: RSVP to meetups with waitlist support
- **Image Upload**: Upload images for meetups
- **Responsive Design**: Mobile-first design with Tailwind CSS

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- React Router DOM
- React Hook Form + Zod
- Axios
- React Hot Toast

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:8080
```
